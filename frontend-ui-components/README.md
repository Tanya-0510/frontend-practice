# 🎨 Frontend UI Components

A collection of reusable and modern frontend UI components built using HTML, CSS, and JavaScript.

## 🚀 Features

* Responsive UI design
* Clean and minimal layout
* Reusable components
* Smooth styling and interactions

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

## 📂 Project Structure

```
frontend-ui-components/
│── index.html
│── style.css/
│── script.js/
```

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Tanya-0510/frontend-practice.git
   ```
2. Open `index.html` in your browser.
